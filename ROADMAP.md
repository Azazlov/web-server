# FileShare Local — Дорожная карта до v1.0

> **Приоритетная цель продукта:** Разворачивание HTTP/HTTPS локального сервера для обмена данными между устройствами и пользователями (в том числе с телефона). Развёртывание должно быть максимально простым и удобным.

## Фаза v0.5 — Простое развёртывание

> **Ключевая фаза для цели продукта.** Развёртывание должно быть в 1 команду и работать с мобильных устройств в локальной сети.

| # | Задача | Описание | Критерий готовности |
|---|--------|----------|---------------------|
| 5.1 | **Автоопределение IP** | При запуске сервер печатает свой LAN-IP (`http://192.168.x.x:80`). QR-код для быстрого подключения с телефона | QR-код в консоли и в админ-панели |
| 5.2 | **HTTPS самоподписанный** | Генерация самоподписанного SSL-сертификата при первом запуске. Режим `--https`. Поддержка mkcert для dev | `https://<IP>` работает без ошибок браузера (или с 1 кликом доверия) |
| 5.3 | **Production WSGI-сервер** | Заменить `app.run()` на `waitress` (кроссплатформенный). Конфигурация через `config.json` | Сервер держит 50+ одновременных подключений |
| 5.4 | **Docker** | `Dockerfile` (multi-stage, минимальный образ). `docker-compose.yml` с volume для данных | `docker compose up -d` поднимает сервер |
| 5.5 | **Systemd unit** | Файл `fileshare.service` для Linux. `systemctl enable/start/status` | Сервис запускается и рестартует при падении |
| 5.6 | **One-command install** | Единый скрипт установки: pip → deps → config → автозапуск. `curl ... \| sh` для Linux, `.bat` для Windows | Новичок разворачивает за 2 минуты |
| 5.7 | **Zero-config по умолчанию** | Сервер стартует без `config.json`. Автоматический выбор свободного порта (если 80 занят). Вывод URL в консоль | `python app.py` работает из коробки |

## Целевой пользовательский Сценарий v1.0

> **Маша хочет поделиться фотографиями с семьёй.**

1. Маша скачивает `.bat`/`.sh` файл и запускает — сервер поднимается за 30 секунд
2. В консоли появляется QR-код → Маша сканирует телефоном → открывается `https://192.168.1.42`
3. Маша регистрируется (становится админом), загружает фото через drag & drop
4. Семья подключается по ссылке, просматривает фото прямо в браузере (предпросмотр)
5. Маша создаёт публичную ссылку на папку и отправляет бабушке в WhatsApp
6. Бабушка открывает ссылку без регистрации и скачивает фото

**Всё это работает в v1.0.**
